# 🌿 EcoTrack - Waste Management & Complaint System

EcoTrack is a full-stack **MERN application** designed to streamline waste reporting, complaint management, and civic issue tracking between users, staff, and admins.

It helps citizens report waste issues, enables staff to resolve them, and gives admins full control over monitoring and analytics.

---

⭐ Project Goal

To build a smart civic system that improves cleanliness, accountability, and fast issue resolution in urban areas.

## 🚀 Features

### 👤 User Features

- Register / Login (JWT Auth)
- Report waste complaints with images
- Track complaint status
- Raise helpdesk tickets
- View personal dashboard (stats + recent complaints)

### 🧑‍🔧 Staff Features

- View assigned complaints
- Update complaint status (Pending → In Progress → Resolved)
- Manage helpdesk issues
- View performance stats

### 🛠️ Admin Features

- Dashboard analytics (total, pending, resolved, users, staff)
- Assign complaints to staff
- View all complaints & tickets
- Monitor system activity
